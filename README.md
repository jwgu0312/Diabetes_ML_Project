# Diabetes_ML_Project
## Dataset
This project utilizes the BRFSS 2015 (Behavioral Risk Factor Surveillance System) annual survey dataset from the CDC, containing 253,680 samples and 21 health-related behavioral, lifestyle, and sociodemographic features.

## Objective
To develop a multiclass machine learning pipeline that classifies individuals into three categories: Healthy (0), Prediabetes (1), or Diabetes (2). This provides empirical predictive evidence for public health risk screening.

## Project Pipeline
### Notebook 01: Data Inspection & EDA
Imported the raw CDC BRFSS dataset and verified data integrity. Identified and removed 23,899 completely duplicated records to eliminate statistical redundancy and prevent evaluation bias. Plotted feature correlation matrices and generated target class distributions to establish a clean data baseline.

### Notebook 02: Stratified Split, Standardization & SMOTE
Divided the dataset into training and testing sets at an 8:2 ratio using stratified sampling to maintain identical class proportions across both splits. Applied StandardScaler to all 21 predictors for dimensionless scaling. Implemented SMOTE strictly on the training set to mitigate the severe class imbalance where prediabetes samples accounted for only 1.83% of the data.

### Notebook 03: Baseline Models Pipeline Training
Structured a standardized machine learning pipeline to couple preprocessing and algorithm training, ensuring strict reproducibility. Evaluated 6 mandatory baseline models: Logistic Regression, Decision Tree, Random Forest, XGBoost, LightGBM, and Support Vector Machine (SVM).

### Notebook 04: Systematic Hyperparameter Tuning
Conducted systematic hyperparameter optimization for Random Forest, XGBoost, and LightGBM using RandomizedSearchCV paired with 3-fold cross-validation. Locked the optimization search objective on the Macro F1-score to enforce balanced class-specific performance over raw accuracy.

### Notebook 05: Heterogeneous Stacking Ensemble
Designed a two-stage StackingClassifier architecture to combine distinct model behaviors. Extracted the class prediction probabilities generated by the optimized first-layer tree models as meta-features, and fed them into a Logistic Regression meta-model for final decision fusion.

### Notebook 06: Global Model Explainability via SHAP
Integrated the SHAP framework to interpret the internal mechanics of the optimal model. Computed Shapley values to rank global feature contributions, identifying General Health Assessment (GenHlth), Age, High Blood Pressure (HighBP), BMI, and High Cholesterol (HighChol) as the top 5 clinical risk predictors.

### Notebook 07: Final Evaluation & Model Selection
Aggregated performance metrics from all baseline, tuned, and ensemble models into a consolidated metrics table. Prioritized Macro F1-score due to the data imbalance, selecting the Decision Tree as the optimal deployment model with a Macro F1-score of 0.451.
